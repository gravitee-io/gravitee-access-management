What is SPIFFE?

SPIFFE (Secure Production Identity Framework For Everyone) issues short-lived workload identities as JWT-SVIDs. When SPIFFE is enabled, applications in this domain can use the spiffe_jwt token-endpoint authentication method: clients present a JWT-SVID instead of a client secret, and AM verifies it against a configured Trust Domain bundle.

The settings on this page are domain-wide guards (SSRF protection, JWKS fetch limits, validation policy). Per-trust-domain JWKS sources and refresh intervals live under Security → Trust Domains.

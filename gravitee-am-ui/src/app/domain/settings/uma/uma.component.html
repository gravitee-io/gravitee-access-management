User-Managed Access (UMA) 2.0 Support Enable resource owner to grant access to his resource to a third requesting party through UMA policies. UMA 2.0 service provider's configuration is available under /uma/.well-known/uma2-configuration.

UMA 2.0 is a federated authorization standard protocol built on top of OAuth 2.0 which enables party-to-party sharing.

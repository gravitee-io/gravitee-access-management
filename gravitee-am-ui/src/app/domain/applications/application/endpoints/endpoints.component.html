Endpoints
Authorization endpoint The authorization endpoint is used to obtain an OAuth 2.0 token using either authorization code or implicit grant type Token endpoint The token endpoint is used to obtain an OAuth 2.0 token Introspection endpoint The introspection endpoint allows to access information about an OAuth 2.0 token Revocation endpoint The revocation endpoint allows to invalidate a previously obtained OAuth 2.0 refresh or access token UserInfo endpoint The authorization endpoint is used to obtain an OAuth 2.0 token using either authorization code or implicit grant type End session endpoint The end session endpoint allows to invalidate a user's session OpenID Configuration endpoint The OpenId configuration endpoint allows third party systems to get the complete OpenId configuration SCIM Configuration endpoint The SCIM configuration endpoint allows third party systems to get the complete SCIM configuration

Useful endpoints

All necessary endpoints to communicate with the Authorization Server.
